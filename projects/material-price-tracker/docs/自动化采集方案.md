## 安装步骤

### 1. Windows端配置

#### 安装Python依赖
```cmd
pip install selenium webdriver-manager
```

#### 测试采集脚本
```cmd
cd \\wsl$\Ubuntu\home\hh127\material-price-tracker
python windows_auto_collector.py
```

#### 设置定时任务
1. 打开"任务计划程序"（Win+R → taskschd.msc）
2. 创建基本任务
3. 触发器：每天 08:00
4. 操作：启动程序
   - 程序：`python`
   - 参数：`windows_auto_collector.py`
   - 起始于：`\\wsl$\Ubuntu\home\hh127\material-price-tracker`

### 2. WSL端配置

#### 设置Cron任务
```bash
crontab -e
```

添加以下内容：
```bash
# 每天早上8:30自动导入价格数据
30 8 * * * cd /home/hh127/material-price-tracker && /home/hh127/material-price-tracker/.venv/bin/python auto_import.py >> /tmp/auto_import.log 2>&1
```

## 文件说明

| 文件 | 说明 | 运行环境 |
|------|------|----------|
| `windows_auto_collector.py` | Windows自动采集脚本 | Windows |
| `auto_import.py` | WSL自动导入脚本 | WSL |
| `mysteel_cookies.json` | Cookie文件 | 共享 |
| `price_data/` | 本地备份目录 | Windows |
| `imported/` | 已处理文件目录 | WSL |

## 手动操作

### 手动采集（Windows）
```cmd
python windows_auto_collector.py
```

### 手动导入（WSL）
```bash
python auto_import.py
```

### 查看日志
```bash
tail -f /tmp/auto_import.log
```

## 添加更多城市

编辑 `windows_auto_collector.py`，在 `CONFIG['urls']` 中添加：

```python
'urls': {
    '西安': 'https://jiancai.mysteel.com/m/...',
    '北京': 'https://jiancai.mysteel.com/m/...',
    '上海': 'https://jiancai.mysteel.com/m/...',
}
```

## 故障排除

### Cookie过期
1. 在浏览器中重新登录我的钢铁网
2. 更新 `mysteel_cookies.json`

### Chrome版本不匹配
```cmd
pip install webdriver-manager --upgrade
```

### WSL目录不可访问
确保Windows可以访问 `\\wsl$\Ubuntu\home\hh127\material-price-tracker`
