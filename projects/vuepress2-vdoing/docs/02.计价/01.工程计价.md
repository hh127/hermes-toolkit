# 工程计价

工程计价是建设工程造价管理的核心内容，主要包括定额计价和清单计价两种方式。

## 计价方式

### 定额计价

定额计价是传统的计价方式，依据国家或地方发布的定额标准进行计价。

### 清单计价

清单计价是国际通行的计价方式，依据《建设工程工程量清单计价规范》进行计价。

## 造价构成

| 费用项目 | 说明 |
|---------|------|
| 分部分项工程费 | 按清单项目计算的工程费用 |
| 措施项目费 | 施工过程中发生的非实体项目费用 |
| 其他项目费 | 暂列金额、暂估价、计日工等 |
| 规费 | 社会保险费、住房公积金等 |
| 税金 | 增值税等 |

## 参考资料

- 《建设工程工程量清单计价规范》(GB 50500-2013)
- 《建筑安装工程费用项目组成》(建标[2013]44号)
