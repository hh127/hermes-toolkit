# 关于

这是一个个人Wiki知识库，主要用于记录工程造价、合约管理、法律法规等专业知识。

## 内容分类

- **计量** - 工程计量相关知识，包括工程量计算规则、清单编制等
- **计价** - 工程计价相关知识，包括定额计价、清单计价、造价分析等
- **法规** - 法律法规标准库，涵盖现行法律、行政法规、部门规章等

## 技术栈

- [VuePress 2.x](https://v2.vuepress.vuejs.org/) - 静态网站生成器
- [vdoing2](https://github.com/hh127/vuepress2-vdoing) - 基于 vdoing 主题移植到 VuePress 2.x
